# 键盘连点器使用说明

## 功能介绍

本程序是一个多功能键盘连点器，具有以下特性：
1. **单个按键连点** - 可以连续按下同一个按键
2. **按键序列连点** - 可以按照设定的顺序连续按下多个按键
3. **录制回放** - 可以录制用户的按键操作并重复播放
4. **自定义热键** - 可以设置开始和停止的快捷键
5. **参数调节** - 可以调整按键间隔时间和重复次数
6. **配置保存** - 自动保存设置，下次启动时自动加载

## 使用方法

### 启动程序
双击运行 `Keyboard.py` 文件，或者在命令行中执行：
```bash   ”“bash
python Keyboard.py
```

### 界面说明

#### 工作模式
- **单个按键连点**：适用于需要重复按同一个键的场景
- **按键序列连点**：适用于需要按特定顺序按键的场景
- **录制回放**：适用于复杂的按键操作序列

#### 参数设置
1. **间隔时间**：设置每次按键之间的间隔（单位：秒）
2. **点击次数**：设置重复次数（0表示无限循环）
3. **热键设置**：可以自定义开始和停止的快捷键

### 各模式详细使用说明

#### 单个按键连点模式
1. 在"按键"输入框中输入要连点的键名（如：a, space, enter等）
2. 设置间隔时间和点击次数
3. 点击"开始"按钮或按设置的开始热键开始连点
4. 点击"停止"按钮或按设置的停止热键停止连点

#### 按键序列连点模式
1. 在文本框中每行输入一个按键名称
2. 程序会按顺序执行这些按键操作
3. 设置间隔时间和点击次数
4. 点击"开始"按钮或按设置的开始热键开始连点
5. 点击"停止"按钮或按设置的停止热键停止连点

#### 录制回放模式
1. 点击"开始录制"按钮开始录制按键操作
2. 执行你想要录制的按键操作
3. 点击"停止录制"按钮结束录制
4. 点击"播放录制"按钮回放录制的内容
5. 可以通过"清空录制"按钮清除已录制的内容

### 热键设置
1. 点击"开始热键"或"停止热键"按钮
2. 在弹出的对话框中按下你想要设置的按键
3. 程序会自动保存新的热键设置

## 注意事项

1. 程序会自动保存配置，下次启动时会加载上次的设置
2. 如果需要停止无限循环，请使用设置的停止热键或点击"停止"按钮
3. 某些游戏可能会屏蔽程序的按键操作
4. 请合理使用此工具，避免在不适当的情况下使用

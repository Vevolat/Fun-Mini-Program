# 键盘连点器

这是一个基于Python和Tkinter开发的图形化键盘连点器工具，支持多种键盘自动化功能。

## 功能特性

1. **单个按键连点** - 可以连续按下一个指定的按键
2. **按键序列连点** - 可以按照设定的序列连续按下多个按键
3. **录制回放** - 可以录制用户的按键操作并进行回放
4. **自定义间隔时间** - 可以设置按键之间的间隔时间
5. **自定义按键次数** - 可以设置按键重复的次数（支持无限循环）
6. **热键控制** - 支持自定义开始和停止热键
7. **配置保存** - 自动保存和加载用户配置

## 安装依赖

```bash
pip install -r requirements.txt
```

或者单独安装：

```bash
pip install keyboard pyautogui
```

## 使用方法

1. 运行程序：
   ```bash
   python Keyboard.py
   ```

2. 选择工作模式：
   - **单个按键连点**：在输入框中输入要连点的按键，如 `a`、`space`、`enter` 等
   - **按键序列连点**：在文本区域中每行输入一个按键，程序会按顺序连点
   - **录制回放**：点击"开始录制"按钮录制按键操作，录制完成后可点击"播放录制"进行回放

3. 设置参数：
   - 调整间隔时间（秒）
   - 设置点击次数（0表示无限循环）

4. 设置热键：
   - 点击热键按钮可自定义开始和停止热键

5. 开始使用：
   - 点击"开始"按钮或按开始热键启动连点
   - 点击"停止"按钮或按停止热键停止连点
